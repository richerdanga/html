Detalle del proyecto program pos

Este proyecto se basa en un software para facturar y para llevar el inventario en pequeños negocios.

Tiene posibilidad de imprimir facturas por medio de una impresora termica, se puede conectar un lector de codigo e barras

    El programa comprende los siguientes modulos

  • Usuarios
  • Clientes
  • Productos
  • Ventas
  • Inventarios
  • Reportes
  • Sesion
